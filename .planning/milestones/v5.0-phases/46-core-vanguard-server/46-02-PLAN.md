    2. `server/vanguard/server.go` — Server struct and lifecycle:

       Server struct fields:
       ```go
       type Server struct {
           config             Config
           httpServer         *http.Server
           container          *di.Container
           grpcServer         *grpc.Server   // resolved from DI, the *grpc.Server from server/grpc
           logger             *slog.Logger
       ```

       `NewServer(cfg Config, logger *slog.Logger, container *di.Container, grpcSrv *grpc.Server) *Server` — Constructor. Resolves `*health.Manager` optionally from DI container. Accepts `*grpc.Server` directly (resolved by module provider from `server/grpc.Server.GRPCServer()`).

       `SetUnknownHandler(h http.Handler)` — Sets user-defined handler for non-RPC routes. Called before OnStart.

       `OnStart(ctx context.Context) error` — Full lifecycle:
       1. Discover Connect services: `di.ResolveAll[connect.ConnectRegistrar](s.container)`
       2. Build `[]*vanguard.Service` from each ConnectRegistrar's `RegisterConnect()` via `vanguard.NewService(path, handler)`
       3. Add reflection handlers if `s.config.Reflection`:
          - Collect ALL service names: from Connect registrars (path → service name) and from gRPC server (`s.grpcServer.GetServiceInfo()`)
          - Create `grpcreflect.NewStaticReflector(serviceNames...)`
          - Create v1 and v1alpha handlers: `grpcreflect.NewHandlerV1(reflector)` and `grpcreflect.NewHandlerV1Alpha(reflector)`
          - Add both as `vanguard.NewService(path, handler)` to the services slice
       4. Bridge gRPC server: `vanguardgrpc.NewTranscoder(s.grpcServer)` → append to services
       5. Build unknown handler:
          - If health.Manager present, create health mux via `buildHealthMux(s.healthManager)`
          - If user unknown handler set, compose: health mux handles /healthz /readyz /livez, user handler handles everything else
          - If only health, health mux is the unknown handler
          - If only user handler, user handler is the unknown handler
          - Use `vanguard.WithUnknownHandler(composedHandler)` if any handler exists
       6. Create transcoder: `vanguard.NewTranscoder(services, opts...)`
       7. Configure h2c `http.Server`:
          ```go
          p := new(http.Protocols)
          p.SetHTTP1(true)
          p.SetUnencryptedHTTP2(true)

          s.httpServer = &http.Server{
              Addr:              fmt.Sprintf(":%d", s.config.Port),
              Handler:           transcoder,
              Protocols:         p,
              ReadTimeout:       s.config.ReadTimeout,
              WriteTimeout:      s.config.WriteTimeout,
              ReadHeaderTimeout: s.config.ReadHeaderTimeout,
              IdleTimeout:       s.config.IdleTimeout,
          }
          ```
       8. Start serving in goroutine. Log server started with port, service count, reflection, health status.

       `OnStop(ctx context.Context) error` — Graceful shutdown:
       1. Call `s.httpServer.Shutdown(ctx)` — gracefully drains connections
       2. Log server stopped

    3. `server/vanguard/module.go` — Module registration following `server/grpc/module.go` pattern:

       ```go
       func resolveLogger(c *gaz.Container) *slog.Logger { ... }  // Same pattern as grpc

       func provideConfig(defaultCfg Config) func(*gaz.Container) error { ... }  // Same pattern

       func provideServer(c *gaz.Container) error {
           // Eager provider
           return gaz.For[*Server](c).
               Eager().
               Provider(func(c *gaz.Container) (*Server, error) {
                   cfg, err := gaz.Resolve[Config](c)
                   // Resolve *grpc.Server from gRPC Server wrapper
                   grpcSrv, err := gaz.Resolve[*grpc.Server](c)
                   // ... or resolve the grpc.Server wrapper and call GRPCServer()
                   return NewServer(cfg, resolveLogger(c), c, grpcSrv), nil
               })
       }

       func NewModule() gaz.Module {
           defaultCfg := DefaultConfig()
           return gaz.NewModule("vanguard").
               Flags(defaultCfg.Flags).
               Provide(provideConfig(defaultCfg)).
               Provide(provideServer).
               Build()
       }
       ```

       **DI resolution for gRPC server:** The provider needs the `*grpc.Server` from the gRPC module. Resolve `*grpc.Server` via the gRPC `Server` wrapper: `gaz.Resolve[*grpc.Server](c)` — BUT the gRPC module registers `*grpc.Server` as the wrapper `Server` type, not the raw `*grpc.Server`. So instead:
       - Resolve `*grpcpkg.Server` (the gaz wrapper), then call `.GRPCServer()` to get `*grpc.Server`
       - Pass the raw `*grpc.Server` to `NewServer`
       - This matches the CONTEXT.md decision: "Vanguard server resolves `*grpc.Server` from DI"
