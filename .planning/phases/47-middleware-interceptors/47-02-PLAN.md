4. **Create `server/vanguard/middleware.go`:**

   **TransportMiddleware interface:**
   ```go
   type TransportMiddleware interface {
       Name() string
       Priority() int
       Wrap(http.Handler) http.Handler
   }
   ```

   **Priority constants:**
   ```go
   const (
       PriorityCORS = 0    // CORS runs first (outermost handler).
       PriorityOTEL = 100  // OTEL wraps after CORS.
   )
   ```

   **collectTransportMiddleware()** — takes `*di.Container` and `*slog.Logger`. Uses `di.ResolveAll[TransportMiddleware]`, sorts by priority, applies in reverse order (lowest priority wraps outermost), returns wrapped handler. Signature: `func collectTransportMiddleware(container *di.Container, logger *slog.Logger, handler http.Handler) http.Handler`.

   **CORSMiddleware** — implements TransportMiddleware. Takes `CORSConfig` and `devMode bool`. In dev mode: uses `cors.AllowAll()`. In production: uses `cors.New(cors.Options{...})` with configured fields. `Wrap()` calls `corsHandler.Handler(next)`. Priority = PriorityCORS (0).

   **OTELMiddleware** — implements TransportMiddleware. Takes `*sdktrace.TracerProvider`. `Wrap()` uses `otelhttp.NewMiddleware("vanguard", otelhttp.WithTracerProvider(tp), otelhttp.WithFilter(...))`. Filter excludes `/healthz`, `/readyz`, `/livez`, and reflection paths (`/grpc.reflection.`). Priority = PriorityOTEL (100).

   **OTELConnectBundle** — implements `connect.ConnectInterceptorBundle` (from `server/connect/`). Takes `*sdktrace.TracerProvider`. `Interceptors()` calls `otelconnect.NewInterceptor(otelconnect.WithTracerProvider(tp), otelconnect.WithFilter(...))` — handle the error return (log warning and return empty slice on error). Filter excludes health check procedures. Priority = PriorityValidation - 1 (99) — runs before validation but after auth. Name = "otelconnect".
