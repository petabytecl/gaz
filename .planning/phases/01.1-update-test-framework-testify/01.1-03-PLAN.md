**Import organization:**
```go
import (
    "errors"
    "strings"
    "testing"
    
    "github.com/stretchr/testify/assert"
    "github.com/stretchr/testify/require"
    "github.com/stretchr/testify/suite"
)
```

**Assertion conversions:**
- `if c == nil { t.Fatal(...) }` → `require.NotNil(s.T(), c)`
- `if c.services == nil { t.Fatal(...) }` → `require.NotNil(s.T(), c.services)`
- `if c.built { t.Fatal(...) }` → `assert.False(s.T(), c.built)`
- `t.Fatalf("...failed: %v", err)` → `require.NoError(s.T(), err)`
- `if err != nil { t.Fatalf(...) }` → `require.NoError(s.T(), err)`
- `if !strings.Contains(err.Error(), "xxx")` → `assert.Contains(s.T(), err.Error(), "xxx")`
- `if pool1.id != 1 { t.Errorf(...) }` → `assert.Equal(s.T(), 1, pool1.id)`
- `if pool1 != pool2 { t.Error(...) }` → `assert.Same(s.T(), pool1, pool2)` for pointer equality

**Pattern for Build() tests:**
```go
func (s *ContainerSuite) TestBuild_Idempotent() {
    c := New()
    require.NoError(s.T(), c.Build())
    require.NoError(s.T(), c.Build()) // second call also succeeds
}
```

**Pattern for error message assertions:**
```go
func (s *ContainerSuite) TestBuild_EagerError_PropagatesWithContext() {
    c := New()
    For[*testFailingService](c).Eager().Provider(func(c *Container) (*testFailingService, error) {
        return nil, errors.New("startup failed")
    })
    
    err := c.Build()
    require.Error(s.T(), err)
    assert.Contains(s.T(), err.Error(), "testFailingService")
    assert.Contains(s.T(), err.Error(), "startup failed")
}
```

Keep all helper types (testEagerPool, testFailingService, etc.) at package level.
Keep concurrent test patterns (sync.WaitGroup, goroutines), just update assertions.
