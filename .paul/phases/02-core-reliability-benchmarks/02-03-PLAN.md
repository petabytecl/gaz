<objective>
## Goal
Fix EventBus synchronization: snapshot-then-deliver for Publish, release lock before drain in unsubscribe, context-bounded CloseWithContext.

## Purpose
The highest-convergence finding in the audit (4 agents). Publish holds RLock during blocking channel sends, causing shutdown hangs. unsubscribe holds write lock during handler drain.

## Output
Modified eventbus/bus.go with fixed synchronization. New tests for concurrent scenarios.
</objective>


## AC-2: unsubscribe releases lock before draining handler
```gherkin

```

## AC-3: CloseWithContext respects deadline
```gherkin

```

## AC-4: No performance regression vs baseline

