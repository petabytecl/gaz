<objective>
## Goal
Unify Cobra Start() and Run() startup paths by extracting shared lifecycle logic. The strongest finding in the AEGIS audit (upheld by Devil's Advocate).

## Purpose
Every Cobra-based application silently loses worker supervision, parallel startup, and panic recovery. This affects every gaz consumer using WithCobra().

## Output
Shared startLifecycleServices() method. Start() delegates to it. doStop() uses cached startup order. New integration tests.
</objective>
