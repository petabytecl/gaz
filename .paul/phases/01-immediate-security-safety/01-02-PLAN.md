<objective>
## Goal
Add defensive safety guards to the DI container: `built` flag check in Register/ReplaceService, checked type assertion in ResolveAll[T], nil guard in instanceServiceAny.ServiceType().

## Purpose
Enforce the ErrAlreadyBuilt contract that was defined but never checked (phantom API contract). Prevent panics from unchecked type assertions and nil ServiceType. These are trivial fixes with near-zero regression risk that harden the DI container for external consumers.

## Output
Modified di/container.go, di/resolution.go, di/service.go, di/registration.go with safety guards. New tests in di/container_test.go and di/resolution_test.go.
</objective>
